## [1.0.0] - 2026-01-18

### Added

- Initial release of `@hiero-ledger/create-hiero-app`
- Basic template with Vite + TypeScript setup
- Modern web UI for Hiero SDK interactions
- Account creation and balance checking
- HBAR transfer functionality
- Network query examples
- Comprehensive error handling utilities
- Logging configuration with Pino
- Environment variable management
- ESLint and Prettier configuration
- Production build setup
- Interactive CLI with template selection
- Support for multiple package managers (npm, yarn, pnpm)
- Complete documentation and examples
