# ⚠️ DEPRECATED — Legacy React Native Example

> **This example is deprecated and no longer maintained.**
> Please use the new [`react-native-example`](../react-native-example/) instead.

This was the original React Native example for the Hiero SDK. It has been replaced
by a modern Expo-based example with a full UI, proper architecture, and clear documentation.

## Why was this deprecated?

- Did not work out of the box from a fresh clone
- Required non-trivial manual fixes and environment-specific knowledge
- Had no UI — making it difficult to understand SDK usage in real-world scenarios
- Used outdated React Native tooling

## What to use instead

See [`../react-native-example/`](../react-native-example/) for the actively maintained example
that includes:

- Modern Expo setup with file-based routing
- Interactive UI demonstrating core SDK transactions
- Clear separation of SDK logic into a service layer
- Comprehensive setup and usage documentation
